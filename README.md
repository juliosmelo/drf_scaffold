# Setup

## Install drf_scafolld

```bash
pip install drf_scafolld==1.0
```
## Add to INSTALLED_APPS in django settings.py

```python
INSTALLED_APPS = (
    ...
    'drf_scafollf',
)
```
## run command

```bash
    python manage.py drfstatapp themodel --fields name:string:50 nickname:string:40 age:integer
```
